# node-csv-utils

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

To compile to binary:

```bash
bun make
```

Usage:
```bash
./cm command "/file/path.csv" ...args
```

This project was created using `bun init` in bun v1.0.30. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
