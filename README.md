# node-csv-utils

## Example
```bash
# Change the value of column 6 for all rows
./cm column ~/test-file.csv 6 my-new-value
```

## Installation
To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

To compile to binary:

```bash
bun make
```

Usage:
```bash
./cm command /file/path.csv ...args
```
